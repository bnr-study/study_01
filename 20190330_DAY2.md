# 연산자
- 다항연산자
  - +,-,*,/,%
- 단항연산자
  - ++
  - --
  
# switch
```
  switch(region){
    case '송파':
      console.log('송파');
      break;
    case '강남':
      console.log('강남');
      break;
    default:
      break;
  }
```

# for
```
  for( var i=0; i<10; i++ ){
    console.log(i);
  }
```

# while
```
  var a = 0;
  while(a < 10){
    console.log(a);
    a++;
  }
```
